Como calcular média no JavaScript?

Neste exemplo, vamos primeiramente armazenar o nome de um aluno na variável "nome", desta forma: var nome = "Patrícia"

Observe que o nome está entre aspas, porque é uma string, ou seja, textos, caracteres ou números que queremos printar na tela, devem ser escritos entre aspas.

Depois, é necessário colocar a nota do aluno em relação aos bimestres.

Escrevemos as variáveis que receberão as notas dos alunos, no estilo Camel Case (imagine um camelo cujas corcovas tem altos e baixos), ou seja, em relação às palavras, a primeira deverá ser escrita com a primeira letra em minúsculo, a segunda palavra e as subsequentes, deverão ser escritas com a primeira letra em maiúsculo. Desta forma: notaPrimeiroBimestre.

Agora, precisamos fazer a conta, ou seja, a média da nota dos alunos. Para isso, segundo a matemática, somamos as notas e dividimos por 4, conforme o exemplo, por termos 4 notas bimestrais. Porém essa conta tem uma ordem de precedência e igual na matemática, a primeira coisa que é resolvida, é aquilo que está entre parênteses. Portanto, segue esta parte da solução: var notaFinal = (notaPrimeiroBimestre + notaSegundoBimestre + notaTerceiroBimestre + notaQuartoBimestre) / 4

Então, vamos solicitar ao sistema que printe na tela a média do aluno, utilizando o console.log. Na frente, entre parênteses, colocam-se aspas e dentro delas, temos a string "Aprovado, passou de ano, " repare que deixamos um espaço para que os termos não fiquem grudados ao sair no Console. Depois, colocamos o sinal de mais (+) em seguida, a variável nome, fechando parênteses e colocando ponto e vírgula no final.

Agora, vamos fazer a Condição if e else.Leiam o seguinte: Se nota final for maior que seis, print na tela "Aprovado, passou de ano, fulano de tal". Caso contrário, print "Reprovado". Isso em código, pode ser explicado da seguinte forma: If significa Se. É necessário colocar entre parênteses a notaFinal e o sinal maior que (>). Depois, escrevemos console.log para printar e entre parênteses a string que queremos escrever, concatenando com o sinal de mais (+) a informação inserida na variável nome.

Else, significa "senão", console.log, ou seja, imprima "Reprovado".

Veremos o código completo mais à frente.

Em seguida, vamos colocar uma informação para que a nota não passe de um número X de casas decimais depois do ponto. Neste caso, queremos apenas uma casa decimal. Para que este parâmetro fique fixado, precisamos colocar o código "toFixed". E por fim, printamos na tela, a notaFixada.

Veja o código completo:

var nome = "Patricia"


var notaPrimeiroBimestre = 9

var notaSegundoBimestre = 7

var notaTerceiroBimestre = 8

var notaQuartoBimestre = 6


var notaFinal = (notaPrimeiroBimestre + notaSegundoBimestre + notaTerceiroBimestre + notaQuartoBimestre) / 4


console.log("Média do aluno é: " + notaFinal)


if (notaFinal > 6) {

console.log ("Aprovado, passou de ano, " + nome);

} else {

console.log ("Reprovado")

}


var notaFixada = notaFinal.toFixed(1)

console.log(notaFixada)
